### What is Slido?


Slido is an interactive platform integrated by default into Webex meetings to enhance audience engagement through live Q&A, polls, and quizzes. It allows meeting hosts to crowdsource questions, conduct polls, and run quizzes to make meetings and webinars more interactive.

### Key Features and Usage:


Slido is available to users with an active Webex meeting host license across all meeting sites in an organization.

Hosts can create, activate, edit, and manage polls and surveys during meetings or webinars.

Participants can answer polls, ask questions, vote on questions, and participate in quizzes.

Hosts can moderate questions, approve or dismiss them before they appear live, and add collaborators to help manage Slido interactions.

Slido supports up to 10,000 attendees for polls and Q&A in webinars, and up to 5,000 participants for quizzes.

Hosts and collaborators sign in to Slido using their Webex credentials, ensuring secure access and integration.

Slido data is stored in regional data centers depending on the organization's location, with default storage in Europe.

Administrators can manage Slido access and user roles via Webex Control Hub and the Slido admin portal.

Slido is not available in Webex for Government and has some limitations, such as lack of support in end-to-end encrypted meetings and breakout sessions.

### Administration and Management:


Slido user accounts are provisioned automatically when hosts use Slido for the first time.

Admins can transfer event ownership and manage user roles within the Slido organization.

Access to Slido can be controlled at the organizational, group, or user level through Webex meeting settings.

External guests can be enabled as co-hosts in Slido settings to collaborate on managing polls and Q&A.

Data retention policies for Slido are separate from Webex and can be customized by contacting Slido support.

### Technical Requirements:


Webex Meetings version 41.6 or later and Webex Webinars version 41.9 or later are required.

Microsoft Edge WebView2 is required on Windows devices for Slido to function properly.

Network firewalls must allow connections to specific Slido and Webex domains.

### Additional Information:


Slido can be accessed directly at https://www.slido.com using Webex login.

Hosts can prepare Slido polls and Q&A in advance or manage them live during meetings.

Attendees can be restricted from starting Slido during meetings by hosts or cohosts.

Slido content is included in meeting recordings only if shared via screen share or integrations with Google Slides or PowerPoint.
